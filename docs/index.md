# Read the Docs Test

Welcome to the test documentation site for the **read-the-docs-test** project.

This project demonstrates how to host documentation on [Read the Docs](https://readthedocs.org) using [MkDocs](https://www.mkdocs.org/) with the [Material theme](https://squidfunk.github.io/mkdocs-material/).

## Quick Start

1. Write your docs in Markdown under `docs/`
2. Configure navigation in `mkdocs.yml`
3. Push to GitHub — Read the Docs builds automatically
